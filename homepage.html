Google OAuth2 Homepage

What will this app do with user data?

This app uses Gmail accounts to verify user identities in order to grant access to the information. The data is used to issue an access token saved only in local storage to access full app functionality. This app will not collect or use user data.

How does this app enhance user functionality?

This app contains events pulled from the Google Calendar API and allows users to access and sort this information in a user-friendly way. Only signed in users can access the data in the app.
